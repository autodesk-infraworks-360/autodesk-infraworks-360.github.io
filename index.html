What exactly does the InfraWorks 360 API do?

The REST API allow access to models, proposals and feature data hosted on InfraWorks 360 cloud.

To use the API it's required to, first, create a model using InfraWorks 360 desktop and then sync to Autodesk Cloud, then the model will be available for sharing with the REST API.

REST API - Read data from models

Using the web standard REST protocol, you can read InfraWorks 360 model data, starting with models, each proposal and all the features.

Here a general overview of the API: it access the model information hosted/synched on Autodesk 360 Cloud just like the deskop or the table (iPad) version.
